Semantic Markup

that help the browser and other tools(e.g. accessibility) to determine important structural information about the document. There are some special semantic markup elements which are:
<nav> contains inforamtion relate to the navigation (a menu, index, links etc)..
<header> introductory material at the top of a document
<main> the main content of the document
<article>
<section>
<footer>end material (author, copyright, links)

Block level & Inline elements

Inline level elements do not create a block and allow contents to run with them...
